Todas las librerias se intalan mediante pip3, para esto al instalar python debe de estar en el path de aplicaciones
para que reconozca el comando en consola. Recomiendo que el python lo instalen desde la pagina oficial y no desde
la microsoft store, solo por los directorios en los que se instala cada uno.

>--------------------------------------------------Link de descarga-----------------------------------------------<
https://www.python.org/downloads/   //Yo uso la version 3.10.7
